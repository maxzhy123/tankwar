# Jave学习过程中做的坦克大战小游戏

---

### tankwar_v1.0_bySwing

- 具体功能
  - 通过swing画出坦克模型以及窗口。
  - 通过线程来保持敌方坦克和子弹的移动。
  - 通过IO流实现存档和读档。
  

### tankwar_v2.0_byLwjgl

- 具体功能
  - 使用lwjgl。
  - 通过每次draw来实现坦克和子弹的移动。
  - 使用面向对象的思想设计改造,定义各种接口,让有相关功能的元素实现一个共同的标记接口,
    在循环体中判断元素是否属于该接口，简化代码，减少冗余。
